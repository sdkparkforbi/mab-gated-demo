# mab-gated-demo

mAb-GATED가 **안정성 노드를 잘 맞히는지 직접 확인**하고, **새 가설 생성을 시뮬레이션**하는 정적 데모 사이트.

## 구성
```
mab-gated-demo/
├── index.html   # 예측 검증 + 가설 생성 데모 (CSS·JS 내장)
├── demo.json    # 실제 KG(32,939 엣지)로 미리 계산한 데모 데이터
└── og.png       # 공유 썸네일
```

## 두 가지 데모
1. **🎯 안정성 예측 검증** — 안정성 노드를 가리고, 이웃 맥락만으로 같은 하위범주 후보 중에서 맞히기. 순위·정답 여부를 즉시 확인.
2. **🔮 가설 생성** — 그래프 구조상 연결될 법하지만 아직 직접 보고되지 않은 인과 후보(link prediction)를 2-hop 매개 경로와 함께 제시.

## 데이터 출처 & 정직성
`demo.json`은 실제 지식그래프(`mab_causal_edges_summary2`, 8,670 노드·32,939 엣지)와
15개 공식 안정성 지표(`mab_review_samples`)로 계산했습니다. 인터랙티브 점수는 그래프
**공기반(co-occurrence) 휴리스틱**으로 모델의 추론 방식을 재현한 것이며, 완전한 **GAT 신경망**은
더 정교해 **Hits@1 84.7%**를 달성합니다(사이트 상단에 실제 성능 표시).

## 배포 (Vercel)
정적 사이트이므로 Import → Deploy. `demo.json`을 갱신하려면 DB에서 재계산 후 교체.
